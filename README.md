# Evolving Agent Ecosystem (Open-Ended Evolution Prototype)

## Concept
This system explores open-ended evolution in multi-agent systems:

- dynamic environment
- multi-dimensional agent genomes
- energy-based survival pressure
- emergent communication signals
- shared memory influencing evolution

## Goals
- behavioral divergence over time
- emergent communication patterns
- survival-driven specialization
- non-trivial population dynamics

## Run

```bash
source .venv/bin/activate
python main.py
```

## Research Directions
- language emergence
- tool creation
- memory-driven adaptation
- culture formation
